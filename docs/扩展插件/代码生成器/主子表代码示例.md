# 主子表代码示例

## 场景说明

以**订单主表 + 订单项子表**为例，展示主子表的完整代码结构。

### 主表：订单表 (sys_order)
- 包含订单基本信息
- 一个订单可以有多个订单项

### 子表：订单项表 (sys_order_item)
- 包含订单项详细信息
- 通过 `order_id` 外键关联到订单主表

---

## 1. 数据库表设计

### 订单主表

```sql
CREATE TABLE sys_order (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '订单ID',
    order_no VARCHAR(50) NOT NULL COMMENT '订单号',
    customer_name VARCHAR(100) COMMENT '客户姓名',
    customer_phone VARCHAR(20) COMMENT '客户电话',
    total_amount DECIMAL(10,2) COMMENT '总金额',
    order_status VARCHAR(20) COMMENT '订单状态',
    order_date DATE COMMENT '订单日期',
    delivery_address VARCHAR(500) COMMENT '配送地址',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    create_by BIGINT COMMENT '创建人',
    update_by BIGINT COMMENT '更新人',
    remark VARCHAR(500) COMMENT '备注',
    status INT DEFAULT 0 COMMENT '状态 0：正常 1：停用'
) COMMENT='订单主表';
```

### 订单项子表

```sql
CREATE TABLE sys_order_item (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '订单项ID',
    order_id BIGINT NOT NULL COMMENT '订单ID（外键）',
    product_id BIGINT COMMENT '商品ID',
    product_name VARCHAR(200) COMMENT '商品名称',
    product_code VARCHAR(100) COMMENT '商品编码',
    quantity INT COMMENT '数量',
    unit_price DECIMAL(10,2) COMMENT '单价',
    subtotal DECIMAL(10,2) COMMENT '小计',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    create_by BIGINT COMMENT '创建人',
    update_by BIGINT COMMENT '更新人',
    remark VARCHAR(500) COMMENT '备注',
    status INT DEFAULT 0 COMMENT '状态 0：正常 1：停用',
    FOREIGN KEY (order_id) REFERENCES sys_order(id)
) COMMENT='订单项子表';
```

### 订单主表 Service 实现（重点）

```java
package com.zl.business.service.impl;

import cn.hutool.core.bean.BeanUtil;
import com.zl.business.domain.Order;
import com.zl.business.domain.OrderItem;
import com.zl.business.dto.OrderDto;
import com.zl.business.dto.OrderItemDto;
import com.zl.business.mapper.OrderItemMapper;
import com.zl.business.mapper.OrderMapper;
import com.zl.business.service.OrderService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.List;

/**
 * 订单主表 Service 实现
 *
 * @author code-generator
 * @date 2026-01-24
 */
@Service
public class OrderServiceImpl implements OrderService {

    @Autowired
    private OrderMapper orderMapper;

    @Autowired
    private OrderItemMapper orderItemMapper;

    @Override
    public List<Order> findAll() {
        return orderMapper.selectAll();
    }

    /**
     * 根据ID查询订单，同时查询所有关联的订单项
     */
    @Override
    public Order findById(Long id) {
        Order order = orderMapper.selectById(id);

        // 🔥 关键：自动查询子表数据
        if (order != null) {
            List<OrderItem> orderItemList = orderItemMapper.selectByOrderId(id);
            order.setOrderItemList(orderItemList);
        }

        return order;
    }

    /**
     * 创建订单，同时保存所有订单项（事务保护）
     */
    @Override
    @Transactional(rollbackFor = Exception.class)
    public Boolean create(OrderDto orderDto) {
        // 1. 保存主表
        Order order = BeanUtil.copyProperties(orderDto, Order.class);
        int result = orderMapper.insert(order);

        // 2. 保存子表数据
        List<OrderItemDto> orderItemList = orderDto.getOrderItemList();
        if (orderItemList != null && !orderItemList.isEmpty()) {
            for (OrderItemDto orderItemDto : orderItemList) {
                OrderItem orderItem = BeanUtil.copyProperties(orderItemDto, OrderItem.class);
                // 🔥 关键：自动设置外键，关联到主表
                orderItem.setOrderId(order.getId());
                orderItemMapper.insert(orderItem);
            }
        }

        return result > 0;
    }

    /**
     * 更新订单，同时更新所有订单项（事务保护）
     */
    @Override
    @Transactional(rollbackFor = Exception.class)
    public Boolean update(OrderDto orderDto) {
        // 1. 更新主表
        Order order = BeanUtil.copyProperties(orderDto, Order.class);
        int result = orderMapper.update(order);

        // 2. 删除原有的子表数据
        orderItemMapper.deleteByOrderId(order.getId());

        // 3. 保存新的子表数据
        List<OrderItemDto> orderItemList = orderDto.getOrderItemList();
        if (orderItemList != null && !orderItemList.isEmpty()) {
            for (OrderItemDto orderItemDto : orderItemList) {
                OrderItem orderItem = BeanUtil.copyProperties(orderItemDto, OrderItem.class);
                orderItem.setOrderId(order.getId());
                orderItemMapper.insert(orderItem);
            }
        }

        return result > 0;
    }

    /**
     * 删除订单，同时删除所有关联的订单项（事务保护）
     */
    @Override
    @Transactional(rollbackFor = Exception.class)
    public Boolean deleteById(Long id) {
        // 1. 先删除子表数据
        orderItemMapper.deleteByOrderId(id);

        // 2. 再删除主表数据
        int result = orderMapper.deleteById(id);
        return result > 0;
    }
}
```

## 6. Controller 接口

```java
package com.zl.business.controller;

import com.zl.common.result.Result;
import com.zl.business.domain.Order;
import com.zl.business.dto.OrderDto;
import com.zl.business.service.OrderService;
import io.swagger.v3.oas.annotations.Operation;
import io.swagger.v3.oas.annotations.tags.Tag;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.List;

/**
 * 订单主表控制器
 *
 * @author code-generator
 * @date 2026-01-24
 */
@RestController
@RequestMapping("/business/order")
@Tag(name = "订单管理", description = "订单管理接口")
@Slf4j
public class OrderController {

    @Autowired
    private OrderService orderService;

    /**
     * 查询所有订单（不包含订单项）
     */
    @GetMapping("/list")
    @Operation(summary = "查询订单列表")
    public Result<List<Order>> list() {
        List<Order> list = orderService.findAll();
        return Result.success(list);
    }

    /**
     * 根据ID查询订单（包含订单项）
     */
    @GetMapping("/{id}")
    @Operation(summary = "根据ID查询订单")
    public Result<Order> getById(@PathVariable Long id) {
        Order order = orderService.findById(id);
        return Result.success(order);
    }

    /**
     * 创建订单（包含订单项）
     */
    @PostMapping
    @Operation(summary = "创建订单")
    public Result<Boolean> create(@RequestBody OrderDto orderDto) {
        Boolean flag = orderService.create(orderDto);
        return Result.success(flag);
    }

    /**
     * 更新订单（包含订单项）
     */
    @PutMapping
    @Operation(summary = "更新订单")
    public Result<Boolean> update(@RequestBody OrderDto orderDto) {
        Boolean flag = orderService.update(orderDto);
        return Result.success(flag);
    }

    /**
     * 根据ID删除订单（级联删除订单项）
     */
    @DeleteMapping("/{id}")
    @Operation(summary = "删除订单")
    public Result<Boolean> delete(@PathVariable Long id) {
        Boolean flag = orderService.deleteById(id);
        return Result.success(flag);
    }
}
```

## 8. 核心特点总结

### ✅ 主表的特点

1. **Domain 包含子表列表**
   ```java
   private List<OrderItem> orderItemList;
   ```

2. **Service 自动处理子表**
   - `findById()` - 自动查询子表数据
   - `create()` - 自动保存子表数据
   - `update()` - 自动更新子表数据（先删除后插入）
   - `deleteById()` - 自动删除子表数据

3. **Controller 提供完整的CRUD接口**
   - 前端只需调用主表接口，不需要单独操作子表

### ✅ 子表的特点

1. **Domain 包含主表对象引用**
   ```java
   private Order order;
   ```

2. **Mapper 包含关联查询方法**
   ```java
   List<OrderItem> selectByOrderId(@Param("orderId") Long orderId);
   int deleteByOrderId(@Param("orderId") Long orderId);
   ```

3. **包含外键字段**
   ```java
   private Long orderId;  // 指向主表的外键
   ```

### ✅ 前端使用要点

1. **创建/更新时**：DTO中必须包含子表列表字段
   ```javascript
   {
       ...主表字段,
       orderItemList: [...子表数据]
   }
   ```

2. **查询时**：自动返回包含子表列表的完整数据

3. **删除时**：自动级联删除所有子表数据

### ✅ 事务保护

所有涉及主子表的操作都使用 `@Transactional` 注解，确保数据一致性：
- 保存失败时，主表和子表都会回滚
- 更新失败时，不会出现数据不一致
- 删除失败时，不会出现孤立数据

## 10. 优势

✅ **前端友好**：一次接口调用完成主子表操作
✅ **事务安全**：自动事务保护，数据一致性有保障
✅ **代码简洁**：不需要手动处理关联关系
✅ **维护方便**：主子表逻辑集中在一个Service中
✅ **扩展灵活**：一个主表可以关联多个子表
