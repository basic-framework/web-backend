# 数据脱敏模块使用示例

## 1. 基本使用示例

### 1.1 实体类定义

```java
package com.zl.model.entity;

import com.zl.sensitive.annotation.Sensitive;
import com.zl.sensitive.core.SensitiveStrategy;
import lombok.Data;

@Data
public class UserInfo {
    private Long id;
    private String username;
    
    @Sensitive(strategy = SensitiveStrategy.PHONE)
    private String phoneNumber;
    
    @Sensitive(strategy = SensitiveStrategy.ID_CARD)
    private String idCard;
    
    @Sensitive(strategy = SensitiveStrategy.EMAIL)
    private String email;
    
    @Sensitive(strategy = SensitiveStrategy.BANK_CARD)
    private String bankCard;
    
    @Sensitive(strategy = SensitiveStrategy.ADDRESS)
    private String address;
    
    @Sensitive(strategy = SensitiveStrategy.SKY)
    private String apiKey;
    
    // 非敏感字段
    private String nickname;
    private Integer age;
    private String gender;
}
```

### 1.2 控制器示例

```java
package com.zl.web.controller;

import com.zl.model.entity.UserInfo;
import com.zl.sensitive.core.SensitiveService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/users")
public class UserController {
    
    @Autowired
    private UserService userService;
    
    @GetMapping("/{id}")
    public Result<UserInfo> getUserInfo(@PathVariable Long id) {
        UserInfo user = userService.getById(id);
        return Result.success(user);
    }
    
    @PostMapping
    public Result<UserInfo> createUser(@RequestBody UserInfo user) {
        UserInfo savedUser = userService.save(user);
        return Result.success(savedUser);
    }
}
```

## 2. 自定义脱敏服务实现

### 2.1 基于角色的脱敏控制

```java
package com.zl.sensitive.service.impl;

import com.zl.common.context.UserThreadLocal;
import com.zl.sensitive.core.SensitiveService;
import org.springframework.stereotype.Service;

@Service
public class RoleBasedSensitiveService implements SensitiveService {
    
    @Override
    public boolean isSensitive() {
        // 获取当前用户信息
        String currentUser = UserThreadLocal.get();
        
        // 如果是管理员，不进行脱敏
        if (isAdmin(currentUser)) {
            return false;
        }
        
        // 如果是数据管理员，可以查看部分敏感信息
        if (isDataManager(currentUser)) {
            return false; // 或者根据具体需求实现部分脱敏
        }
        
        // 普通用户需要进行脱敏
        return true;
    }
    
    private boolean isAdmin(String username) {
        // 实际项目中应该从数据库或权限系统查询
        return "admin".equals(username);
    }
    
    private boolean isDataManager(String username) {
        // 实际项目中应该从数据库或权限系统查询
        return "data_manager".equals(username);
    }
}
```

### 2.2 基于时间的脱敏控制

```java
package com.zl.sensitive.service.impl;

import com.zl.sensitive.core.SensitiveService;
import org.springframework.stereotype.Service;

import java.time.LocalTime;

@Service
public class TimeBasedSensitiveService implements SensitiveService {
    
    @Override
    public boolean isSensitive() {
        // 工作时间（9:00-18:00）不脱敏，其他时间脱敏
        LocalTime now = LocalTime.now();
        return now.isBefore(LocalTime.of(9, 0)) || now.isAfter(LocalTime.of(18, 0));
    }
}
```

### 2.3 基于环境的脱敏控制

package com.zl.sensitive.service.impl;

## 3. 自定义脱敏策略

### 3.1 扩展SensitiveStrategy枚举

```java
package com.zl.sensitive.core;

import cn.hutool.core.util.DesensitizedUtil;
import cn.hutool.core.util.StrUtil;
import lombok.AllArgsConstructor;

import java.util.function.Function;

@AllArgsConstructor
public enum SensitiveStrategy {
    
    // 现有策略...
    ID_CARD(s -> DesensitizedUtil.idCardNum(s, 3, 4)),
    SKY(s -> DesensitizedUtil.idCardNum(s, 0, 1)),
    PHONE(DesensitizedUtil::mobilePhone),
    ADDRESS(s -> DesensitizedUtil.address(s, 8)),
    EMAIL(DesensitizedUtil::email),
    BANK_CARD(DesensitizedUtil::bankCard),
    
    /**
     * 姓名脱敏
     */
    NAME(s -> {
        if (StrUtil.isBlank(s)) {
            return s;
        }
        if (s.length() <= 2) {
            return s.charAt(0) + "*";
        }
        return s.charAt(0) + "*".repeat(s.length() - 2) + s.charAt(s.length() - 1);
    }),
    
    /**
     * 密码脱敏（全部显示为*）
     */
    PASSWORD(s -> "******"),
    
    /**
     * 自定义车牌号脱敏
     */
    LICENSE_PLATE(s -> {
        if (StrUtil.isBlank(s) || s.length() < 3) {
            return s;
        }
        return s.substring(0, 2) + "***" + s.substring(s.length() - 2);
    }),
    
    /**
     * 银行账号脱敏
     */
    BANK_ACCOUNT(s -> {
        if (StrUtil.isBlank(s) || s.length() < 8) {
            return s;
        }
        return s.substring(0, 4) + "****" + s.substring(s.length() - 4);
    });
    
    private final Function<String, String> desensitizer;
    
    public Function<String, String> desensitizer() {
        return desensitizer;
    }
}
```

        // 模拟管理员访问，不需要脱敏

## 5. 高级应用场景

### 5.1 条件性脱敏

```java
package com.zl.sensitive.annotation;

import com.fasterxml.jackson.annotation.JacksonAnnotationsInside;
import com.fasterxml.jackson.databind.annotation.JsonSerialize;
import com.zl.sensitive.core.SensitiveStrategy;
import com.zl.sensitive.handler.ConditionalSensitiveHandler;

import java.lang.annotation.ElementType;
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;
import java.lang.annotation.Target;

/**
 * 条件性脱敏注解
 */
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.FIELD)
@JacksonAnnotationsInside
@JsonSerialize(using = ConditionalSensitiveHandler.class)
public @interface ConditionalSensitive {
    SensitiveStrategy strategy();
    String condition() default ""; // 条件表达式
}
```

### 5.2 多级脱敏

```java
package com.zl.sensitive.core;

import cn.hutool.core.util.StrUtil;
import lombok.AllArgsConstructor;

import java.util.function.Function;

@AllArgsConstructor
public enum MultiLevelSensitiveStrategy {
    
    /**
     * 多级手机号脱敏
     * level 1: 138****5678
     * level 2: 138*****78
     * level 3: 1*******8
     */
    PHONE_MULTI_LEVEL(s -> {
        int level = getDesensitizeLevel();
        return switch (level) {
            case 1 -> DesensitizedUtil.mobilePhone(s);
            case 2 -> s.substring(0, 3) + "*****" + s.substring(s.length() - 2);
            case 3 -> s.charAt(0) + "*******" + s.charAt(s.length() - 1);
            default -> s;
        };
    });
    
    private final Function<String, String> desensitizer;
    
    public Function<String, String> desensitizer() {
        return desensitizer;
    }
    
    private static int getDesensitizeLevel() {
        // 从上下文或配置中获取脱敏级别
        return 1; // 默认级别
    }
}
```

## 6. 最佳实践总结

1. **合理选择脱敏策略**：根据数据类型和业务需求选择合适的脱敏策略
2. **实现灵活的脱敏控制**：基于用户角色、环境、时间等因素动态控制脱敏行为
3. **充分测试脱敏效果**：编写单元测试和集成测试验证脱敏功能
4. **考虑性能影响**：脱敏处理会增加序列化时间，对性能敏感的场景需谨慎使用
5. **注意安全性**：脱敏仅适用于展示层，确保数据库和日志中的敏感数据得到适当保护

通过以上示例，开发者可以快速掌握zl-common-sensitive数据脱敏模块的使用方法，并根据实际业务需求进行灵活扩展。
