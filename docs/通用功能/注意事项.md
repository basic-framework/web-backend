## 代理导致客户端IP无法检测
在Nginx反向代理中，我们会因为代理导致无法把真正的客户端IP传到服务端。
必须配置以下参数（核心：把客户端真实 IP 传递到后端）：

```
server {
    listen 80;
    server_name your-domain.com; # 你的域名/服务器IP

    location / {
        # ========== 核心：传递客户端真实IP到后端 ==========
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr; # 客户端真实IP
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for; # 多级代理IP链
        
        # 反向代理到你的后端服务
        proxy_pass http://127.0.0.1:8080; # 后端服务地址
        proxy_connect_timeout 60s;
        proxy_read_timeout 60s;
    }
}
}
```
X-Real-IP：直接传递客户端真实 IP；

X-Forwarded-For：传递客户端 IP + 代理服务器 IP（多级代理时）。
